Stock Crawler
=============
Stock Crawler for [台灣證券交易所](http://www.twse.com.tw/ch/trading/exchange/MI_INDEX/MI_INDEX.php)

## Requirement

- [Python3](https://www.python.org/)

## Run

```
python StockAll_dbyd.py
```

The Result will be in `stock_all_byday` folder.
